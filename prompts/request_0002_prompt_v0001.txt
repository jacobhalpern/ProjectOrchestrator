# LifePlanner

LifePlanner is a local-first Qt 6 Widgets desktop application for personal planning and management. Phase 1 focuses on a durable desktop foundation: dashboard, calendar, tasks, notes, finance overview, transactions, settings, SQLite persistence, seeded demo data, and test coverage.

## Architecture

- `src/app`: startup and application wiring
- `src/domain`: domain models and enums
- `src/services`: use-case level coordination and dashboard aggregation
- `src/persistence`: SQLite database manager, migrations, and repositories
- `src/ui`: Qt Widgets shell, pages, dialogs, and reusable widgets
- `tests`: Qt Test coverage against an isolated SQLite database

The layering is intentional:

- UI pages call services, not raw SQL.
- Services coordinate repository operations and small business rules.
- Repositories own SQL and persistence mapping.
- Domain models stay Qt-friendly but UI-agnostic.

## Phase 1 Scope

Included in this phase:

- Main application shell with sidebar navigation
- Home dashboard with live database-backed summaries
- Calendar event CRUD
- Task CRUD, filtering, sorting, and completion
- Notes CRUD with pin/unpin
- Finance overview with balances and category summary
- Transaction CRUD with filters
- Settings page with theme/currency/week-start fields
- SQLite initialization, migrations, and seed data
- Repository/service smoke and CRUD tests

Included in the current Phase 2 increment:

- Account CRUD inside the finance area
- Build-tree Qt runtime/plugin deployment for the app and tests
- Windows/MSVC-friendly `NMake Makefiles` presets and instructions

Intentionally deferred:

- Cloud sync
- Mobile clients
- Authentication
- Bank integrations
- Reminders/notifications
- Advanced analytics/reporting
- Budget planning workflows

## Dependencies

- CMake 3.26+
- Qt 6.5+ with `Widgets`, `Sql`, and `Test`
- C++20 compiler
- Windows 11 / Visual Studio 2022 recommended
- Run configure/build from a Visual Studio 2022 Developer Command Prompt

## Configure, Build, Run

Convenience helpers from the repo root:

```powershell
.\build-debug.cmd
.\build-debug-with-assets.cmd
.\run-debug.cmd
```

`build-debug.cmd` configures and builds the current debug setup with the pinned Visual Studio 2022 + Qt 6.10.2 paths.
`build-debug-with-assets.cmd` rebuilds the full debug tree and mirrors `src/assets` into `build/debug/bin/assets`.
`run-debug.cmd` rebuilds first and then launches the app.

Debug with presets:

```powershell
cmake --preset debug
cmake --build --preset build-debug
```

Release with presets:

```powershell
cmake --preset release
cmake --build --preset build-release
```

## Tests

```powershell
ctest --preset test-debug
```

Or manually:

```powershell
ctest --test-dir build/debug --output-on-failure
```

Tests use a temporary isolated SQLite database and do not touch the user database.

## Database Location

By default the production SQLite database is stored in Qt's app data location.

- Windows example: `%LOCALAPPDATA%/<OrganizationName>/<ApplicationName>/lifeplanner.db`

The Settings page displays the resolved path at runtime.

## Migrations And Seed Data

Schema creation and seed insertion are handled through `MigrationRunner`. On first launch, the app creates the schema and inserts sample tasks, notes, accounts, transactions, events, and settings data so the dashboard and pages are immediately useful.

## Extending Later Phases

Recommended Phase 2 directions:

1. Add richer finance workflows such as budgets, bills, and recurring transactions as new services and repositories.
2. Introduce reminder scheduling and notification orchestration without pushing rules into page widgets.
3. Expand dashboard/reporting queries behind services so future desktop and mobile clients can reuse the same core logic.
4. Add import/export and backup features at the persistence/service layer.

{
    if (!m_databaseManager->initialize(errorMessage)) {
        return false;
    }

    const QSqlDatabase database = m_databaseManager->database();

    m_taskService = std::make_unique<TaskService>(TaskRepository(database));
    m_calendarService = std::make_unique<CalendarService>(EventRepository(database));

    QApplication app(argc, argv);
    QApplication::setApplicationName("LifePlanner");

QString normalizedDateDigits(const QString& text)
{
    QString digits = text;
    digits.remove(QRegularExpression("[^0-9]"));
    return digits;
}

QString normalizeImportedText(const QString& text)
{
    return text.simplified().toLower();
}

QString buildImportFingerprint(const QDate& postedDate,
                               double amount,
                               TransactionType transactionType,
                               const QString& description,
                               const std::optional<QString>& memo)
{
    const QString raw = QStringLiteral("%1|%2|%3|%4|%5")
                            .arg(postedDate.toString(Qt::ISODate),
                                 QString::number(amount, 'f', 2),
                                 QString::number(static_cast<int>(transactionType)),
                                 normalizeImportedText(description),
                                 normalizeImportedText(memo.value_or(QString())));
    return QString::fromLatin1(QCryptographicHash::hash(raw.toUtf8(), QCryptographicHash::Sha256).toHex());
}

QString computeStatementFileHash(const QByteArray& content)
{
    return QString::fromLatin1(QCryptographicHash::hash(content, QCryptographicHash::Sha256).toHex());
}

std::optional<QDate> parseFlexibleImportDate(const QString& text)
{
    const QString trimmed = text.trimmed();
    if (trimmed.isEmpty()) {
        return std::nullopt;
    }

    const QString digits = normalizedDateDigits(trimmed);
    if (digits.size() >= 8) {
        const QDate compact = QDate::fromString(digits.left(8), "yyyyMMdd");
        if (compact.isValid()) {
            return compact;
        }
    }

    const QList<QString> formats = {
        QStringLiteral("yyyy-MM-dd"),
        QStringLiteral("M/d/yyyy"),
        QStringLiteral("MM/dd/yyyy"),
        QStringLiteral("M/d/yy"),
        QStringLiteral("MM/dd/yy"),
        QStringLiteral("M-d-yyyy"),
        QStringLiteral("MM-dd-yyyy"),
        QStringLiteral("M-d-yy"),
        QStringLiteral("MM-dd-yy"),
        QStringLiteral("yyyyMMdd")
    };

    for (const QString& format : formats) {
        const QDate parsed = QDate::fromString(trimmed, format);
        if (parsed.isValid()) {
            return parsed;
        }
    }

    return std::nullopt;
}

bool parseCurrencyAmount(const QString& text, double* valueOut)
{
    QString cleaned = text.trimmed();
    if (cleaned.isEmpty()) {
        return false;
    }

    bool negative = false;
    if (cleaned.startsWith('(') && cleaned.endsWith(')')) {
        negative = true;
        cleaned = cleaned.mid(1, cleaned.size() - 2);
    }

    if (cleaned.endsWith('-')) {
        negative = true;
        cleaned.chop(1);
    }

    cleaned.remove('$');
    cleaned.remove(',');
    cleaned.remove(' ');
    cleaned.remove(QRegularExpression("[^0-9.\\-+]"));

    bool ok = false;
    double value = cleaned.toDouble(&ok);
    if (!ok) {
        return false;
    }

    if (negative) {
        value = -qAbs(value);
    }

    if (valueOut != nullptr) {
        *valueOut = value;
    }

    return true;
}

QString detectStatementFormat(const QString& filePath, const QByteArray& content)
{
    const QString suffix = QFileInfo(filePath).suffix().toLower();
    const QString sample = QString::fromUtf8(content.left(4096)).trimmed();

    if (suffix == "ofx" || suffix == "qfx"
        || sample.startsWith("OFXHEADER", Qt::CaseInsensitive)
        || sample.contains("<OFX>", Qt::CaseInsensitive)
        || sample.contains("<STMTTRN>", Qt::CaseInsensitive)) {
        return "OFX/QFX";
    }

    if (suffix == "csv" || (sample.contains(',') && sample.contains('\n'))) {
        return "CSV";
    }

    return {};
}

bool isUnsupportedBinaryStatementFile(const QString& filePath)
{
    const QString suffix = QFileInfo(filePath).suffix().toLower();
    static const QStringList blocked = {
        "pdf",
        "png",
        "jpg",
        "jpeg",
        "gif",
        "bmp",
        "tif",
        "tiff",
        "webp",
        "heic",
        "heif"
    };

    return blocked.contains(suffix);
}

void ensureSqliteDriverAvailable()
{
    if (QSqlDatabase::drivers().contains("QSQLITE")) {
        return;
    }

    const QString appDir = QCoreApplication::applicationDirPath();
    const QStringList candidates = {
        QDir(appDir).filePath("sqldrivers/qsqlited.dll"),
        QDir(appDir).filePath("sqldrivers/qsqlite.dll"),
        QStringLiteral("C:/Qt/6.10.2/msvc2022_64/plugins/sqldrivers/qsqlited.dll"),
        QStringLiteral("C:/Qt/6.10.2/msvc2022_64/plugins/sqldrivers/qsqlite.dll")
    };

    for (const QString& candidate : candidates) {
        if (!QFileInfo::exists(candidate)) {
            continue;
        }

        auto loader = std::make_unique<QPluginLoader>(candidate);
        if (loader->load()) {
            sqlitePluginLoaders().push_back(std::move(loader));
            if (QSqlDatabase::drivers().contains("QSQLITE")) {
                return;
            }
        }
    }
}

DatabaseManager::DatabaseManager(QString databasePath)
    : m_databasePath(std::move(databasePath))
    , m_connectionName(QStringLiteral("lifeplanner_%1").arg(QUuid::createUuid().toString(QUuid::WithoutBraces)))
{
}

DatabaseManager::~DatabaseManager()
{
    if (QSqlDatabase::contains(m_connectionName)) {
        {
            QSqlDatabase db = QSqlDatabase::database(m_connectionName, false);
            if (db.isOpen()) {
                db.close();
            }
        }
        QSqlDatabase::removeDatabase(m_connectionName);
    }
}

bool DatabaseManager::initialize(QString* errorMessage)
{
    if (QSqlDatabase::contains(m_connectionName)) {
        m_databasePath = resolveDatabasePath();
        return true;
    }

    const QString path = resolveDatabasePath();
    ensureSqliteDriverAvailable();
    QFileInfo info(path);
    if (!info.dir().exists() && !info.dir().mkpath(".")) {
        if (errorMessage != nullptr) {
            *errorMessage = QStringLiteral("Unable to create database directory: %1").arg(info.dir().absolutePath());
        }
        return false;
    }

    QSqlDatabase database = QSqlDatabase::addDatabase("QSQLITE", m_connectionName);
    if (!database.isValid()) {
        if (errorMessage != nullptr) {
            *errorMessage = QStringLiteral("QSQLITE driver is not available. Drivers: %1").arg(QSqlDatabase::drivers().join(", "));
        }
        return false;
    }
    database.setDatabaseName(path);
    if (!database.open()) {
        if (errorMessage != nullptr) {
            *errorMessage = database.lastError().text();
        }
        return false;
    }

    if (!MigrationRunner::run(database, errorMessage)) {
        return false;
    }

    m_databasePath = path;
    return true;
}

QSqlDatabase DatabaseManager::database() const
{
    return QSqlDatabase::database(m_connectionName);
}

QString DatabaseManager::databasePath() const
{
    return m_databasePath;
}

QString DatabaseManager::resolveDatabasePath() const
{
    if (!m_databasePath.isEmpty()) {
        return m_databasePath;
    }

    const QString appDataDir = QStandardPaths::writableLocation(QStandardPaths::AppDataLocation);
    return QDir(appDataDir).filePath("lifeplanner.db");
}

bool execStatement(QSqlDatabase& database, const QString& statement, QString* errorMessage)
{
    QSqlQuery query(database);
    if (query.exec(statement)) {
        return true;
    }

    if (errorMessage != nullptr) {
        *errorMessage = query.lastError().text();
    }
    return false;
}

QDateTime nowUtc()
{
    return QDateTime::currentDateTimeUtc();

bool MigrationRunner::run(QSqlDatabase& database, QString* errorMessage)
{
    if (!execStatement(database, "CREATE TABLE IF NOT EXISTS schema_migrations (version INTEGER PRIMARY KEY, name TEXT NOT NULL, applied_at TEXT NOT NULL)", errorMessage)) {
        return false;
    }

    QSqlQuery existingVersions(database);
    if (!existingVersions.exec("SELECT version FROM schema_migrations")) {
        if (errorMessage != nullptr) {
            *errorMessage = existingVersions.lastError().text();
        }
        return false;
    }

    QSet<int> appliedVersions;
    while (existingVersions.next()) {
        appliedVersions.insert(existingVersions.value(0).toInt());
    }

    const auto migrationList = migrations();
    for (const Migration& migration : migrationList) {
        if (appliedVersions.contains(migration.version)) {
            continue;
        }

        if (!database.transaction()) {
            if (errorMessage != nullptr) {
                *errorMessage = database.lastError().text();
            }
            return false;
        }

        bool ok = true;
        for (const QString& statement : migration.statements) {
            if (!execStatement(database, statement, errorMessage)) {
                ok = false;
                break;
            }
        }

        if (ok) {
            QSqlQuery insertMigration(database);
            insertMigration.prepare("INSERT INTO schema_migrations(version, name, applied_at) VALUES(:version, :name, :applied_at)");
            insertMigration.bindValue(":version", migration.version);
            insertMigration.bindValue(":name", migration.name);
            insertMigration.bindValue(":applied_at", nowUtc().toString(Qt::ISODate));
            ok = insertMigration.exec();
            if (!ok && errorMessage != nullptr) {
                *errorMessage = insertMigration.lastError().text();
            }
        }

        if (ok) {
            ok = database.commit();
            if (!ok && errorMessage != nullptr) {
                *errorMessage = database.lastError().text();
            }
        } else {
            database.rollback();
            return false;
        }
    }

    return seedIfNeeded(database, errorMessage);
}

bool MigrationRunner::seedIfNeeded(QSqlDatabase& database, QString* errorMessage)
{
    QSqlQuery countQuery(database);
    if (!countQuery.exec("SELECT COUNT(*) FROM user_settings")) {
        if (errorMessage != nullptr) {
            *errorMessage = countQuery.lastError().text();
        }
        return false;
    }

    if (countQuery.next() && countQuery.value(0).toInt() > 0) {
        return true;
    }

    QSqlQuery insertSettings(database);
    insertSettings.prepare("INSERT INTO user_settings(theme, currency, week_start_day, created_at, updated_at) VALUES(?, ?, ?, ?, ?)");
    insertSettings.addBindValue("Light");
    insertSettings.addBindValue("USD");
    insertSettings.addBindValue(1);
    insertSettings.addBindValue(nowUtc().toString(Qt::ISODate));
    insertSettings.addBindValue(nowUtc().toString(Qt::ISODate));
    if (!insertSettings.exec()) {
        if (errorMessage != nullptr) {
            *errorMessage = insertSettings.lastError().text();
        }
        return false;
    }

    return true;
}

ImportedRowLookup StatementImportRepository::importedRowsForAccount(int accountId) const
{
    ImportedRowLookup lookup;

    QSqlQuery bySourceKey(m_database);
    bySourceKey.prepare(
        "SELECT row.source_row_key, COALESCE(row.created_transaction_id, row.matched_transaction_id) "
        "FROM statement_import_rows row "
        "JOIN statement_imports imports ON imports.id = row.import_id "
        "WHERE imports.target_account_id = ? "
        "AND row.source_row_key IS NOT NULL "
        "AND row.source_row_key <> '' "
        "AND COALESCE(row.created_transaction_id, row.matched_transaction_id) IS NOT NULL");
    bySourceKey.addBindValue(accountId);
    if (bySourceKey.exec()) {
        while (bySourceKey.next()) {
            lookup.transactionIdBySourceRowKey.insert(bySourceKey.value(0).toString(), bySourceKey.value(1).toInt());
        }
    }

    QSqlQuery byFingerprint(m_database);
    byFingerprint.prepare(
        "SELECT row.fingerprint, COALESCE(row.created_transaction_id, row.matched_transaction_id) "
        "FROM statement_import_rows row "
        "JOIN statement_imports imports ON imports.id = row.import_id "
        "WHERE imports.target_account_id = ? "
        "AND row.fingerprint IS NOT NULL "
        "AND row.fingerprint <> '' "
        "AND COALESCE(row.created_transaction_id, row.matched_transaction_id) IS NOT NULL");
    byFingerprint.addBindValue(accountId);
    if (byFingerprint.exec()) {
        while (byFingerprint.next()) {
            lookup.transactionIdByFingerprint.insert(byFingerprint.value(0).toString(), byFingerprint.value(1).toInt());

                                                const std::optional<int>& matchedTransactionId,

{
    const QString sourceRowKey = row.sourceRowKey.isEmpty() ? QString("") : row.sourceRowKey;
    const QString fingerprint = row.fingerprint.isEmpty() ? QString("") : row.fingerprint;
    const QString description = row.description.isEmpty() ? QString("") : row.description;

    QSqlQuery query(m_database);
    query.prepare(
        "INSERT INTO statement_import_rows(import_id, source_row_key, fingerprint, posted_date, description, memo, amount, transaction_type, status, matched_transaction_id, created_transaction_id, created_at) "

    query.addBindValue(createdTransactionId.has_value() ? QVariant(*createdTransactionId) : QVariant());

QList<int> TransactionRepository::createMany(const QList<Transaction>& transactions) const
{
    QList<int> createdIds;
    if (transactions.isEmpty()) {
        return createdIds;
    }

    const QString timestamp = QDateTime::currentDateTimeUtc().toString(Qt::ISODate);
    QSet<int> touchedAccountIds;

    for (const Transaction& transaction : transactions) {
        QSqlQuery query(m_database);
        query.prepare("INSERT INTO transactions(account_id, transaction_date, description, amount, type, category, notes, created_at, updated_at) VALUES(?, ?, ?, ?, ?, ?, ?, ?, ?)");
        query.addBindValue(transaction.accountId);
        query.addBindValue(transaction.transactionDate.toString(Qt::ISODate));
        query.addBindValue(transaction.description);
        query.addBindValue(transaction.amount);
        query.addBindValue(static_cast<int>(transaction.type));
        query.addBindValue(transaction.category);
        query.addBindValue(transaction.notes.has_value() ? QVariant(*transaction.notes) : QVariant());
        query.addBindValue(timestamp);
        query.addBindValue(timestamp);
        if (!query.exec()) {
            return {};
        }

        createdIds.append(query.lastInsertId().toInt());
        touchedAccountIds.insert(transaction.accountId);
    }

    for (int accountId : touchedAccountIds) {
        if (!refreshAccountBalance(accountId)) {
            return {};
        }
    }

    return createdIds;
}

bool TransactionRepository::update(const Transaction& transaction) const
{
    const auto existing = findById(transaction.id);
    QSqlQuery query(m_database);
    query.prepare("UPDATE transactions SET account_id = ?, transaction_date = ?, description = ?, amount = ?, type = ?, category = ?, notes = ?, updated_at = ? WHERE id = ?");
    query.addBindValue(transaction.accountId);
    query.addBindValue(transaction.transactionDate.toString(Qt::ISODate));
    query.addBindValue(transaction.description);
    query.addBindValue(transaction.amount);
    query.addBindValue(static_cast<int>(transaction.type));
    query.addBindValue(transaction.category);
    query.addBindValue(transaction.notes.has_value() ? QVariant(*transaction.notes) : QVariant());
    query.addBindValue(QDateTime::currentDateTimeUtc().toString(Qt::ISODate));
    query.addBindValue(transaction.id);
    const bool ok = query.exec();
    if (ok) {
        refreshAccountBalance(transaction.accountId);
        if (existing.has_value() && existing->accountId != transaction.accountId) {
            refreshAccountBalance(existing->accountId);
        }
    }
    return ok;
}

bool TransactionRepository::remove(int id) const
{
    const auto existing = findById(id);
    QSqlQuery query(m_database);
    query.prepare("DELETE FROM transactions WHERE id = ?");
    query.addBindValue(id);
    const bool ok = query.exec();
    if (ok && existing.has_value()) {
        refreshAccountBalance(existing->accountId);
    }
    return ok;
}

double TransactionRepository::currentMonthTotal(TransactionType type) const
{
    const QDate today = QDate::currentDate();
    const QDate start(today.year(), today.month(), 1);
    const QDate end = start.addMonths(1);
    QSqlQuery query(m_database);
    query.prepare("SELECT COALESCE(SUM(amount), 0) FROM transactions WHERE transaction_date >= ? AND transaction_date < ? AND type = ?");
    query.addBindValue(start.toString(Qt::ISODate));
    query.addBindValue(end.toString(Qt::ISODate));
    query.addBindValue(static_cast<int>(type));
    if (query.exec() && query.next()) {
        return query.value(0).toDouble();
    }
    return 0.0;
}

QMap<QString, double> TransactionRepository::currentMonthExpenseBreakdown() const
{
    const QDate today = QDate::currentDate();
    const QDate start(today.year(), today.month(), 1);
    const QDate end = start.addMonths(1);
    QMap<QString, double> results;
    QSqlQuery query(m_database);
    query.prepare("SELECT category, COALESCE(SUM(amount), 0) FROM transactions WHERE transaction_date >= ? AND transaction_date < ? AND type = ? GROUP BY category ORDER BY SUM(amount) DESC");
    query.addBindValue(start.toString(Qt::ISODate));
    query.addBindValue(end.toString(Qt::ISODate));
    query.addBindValue(static_cast<int>(TransactionType::Expense));
    query.exec();
    while (query.next()) {
        results.insert(query.value(0).toString(), query.value(1).toDouble());
    }
    return results;
}

std::optional<int> TransactionRepository::findPotentialDuplicate(int accountId,
                                                                 const QDate& postedDate,
                                                                 double amount,
                                                                 TransactionType type,
                                                                 const QString& normalizedDescription) const
{
    QSqlQuery query(m_database);
    query.prepare("SELECT id, description FROM transactions WHERE account_id = ? AND transaction_date = ? AND amount = ? AND type = ?");
    query.addBindValue(accountId);
    query.addBindValue(postedDate.toString(Qt::ISODate));
    query.addBindValue(amount);
    query.addBindValue(static_cast<int>(type));

    if (query.exec()) {
        while (query.next()) {
            if (normalizeImportedText(query.value(1).toString()) == normalizedDescription) {
                return query.value(0).toInt();
            }
        }
    }

    return std::nullopt;
}

    query.addBindValue(accountId);
    query.addBindValue(QDateTime::currentDateTimeUtc().toString(Qt::ISODate));
    query.addBindValue(accountId);

ImportPreviewResult FinanceService::previewStatementImport(const QString& filePath, int targetAccountId) const
{
    ImportPreviewResult result;

    if (!m_accountRepository.findById(targetAccountId).has_value()) {
        result.errorMessage = "Choose a destination account before previewing the import.";
        return result;
    }

    QFile file(filePath);
    if (!file.open(QIODevice::ReadOnly)) {
        result.errorMessage = QStringLiteral("Couldn't open %1.").arg(QFileInfo(filePath).fileName());
        return result;
    }

    const QByteArray content = file.readAll();
    const QString sourceFormat = detectStatementFormat(filePath, content);
    if (sourceFormat.isEmpty()) {
        result.errorMessage = isUnsupportedBinaryStatementFile(filePath)
            ? "Unsupported statement file. Export your Wells Fargo transactions as CSV, OFX, or QFX."
            : "Couldn't detect the statement format. Use a Wells Fargo CSV, OFX, or QFX export.";
        return result;
    }

    StatementParseResult parseResult = parserForFormat(sourceFormat).parse(filePath, content);
    if (!parseResult.success) {
        result.errorMessage = parseResult.errorMessage;
        return result;
    }

    ImportPreview preview;
    preview.institution = parseResult.institution;
    preview.sourceFormat = sourceFormat;
    preview.sourceFileName = QFileInfo(filePath).fileName();
    preview.sourceFileHash = computeStatementFileHash(content);
    preview.targetAccountId = targetAccountId;

    const ImportedRowLookup lookup = m_statementImportRepository.importedRowsForAccount(targetAccountId);

    for (int index = 0; index < parseResult.rows.size(); ++index) {
        const StatementParseRow& parsedRow = parseResult.rows.at(index);

        ImportPreviewRow previewRow;
        previewRow.previewIndex = index;
        previewRow.postedDate = parsedRow.postedDate;
        previewRow.description = parsedRow.description.trimmed();
        previewRow.memo = parsedRow.memo;
        previewRow.amount = parsedRow.amount;
        previewRow.transactionType = parsedRow.transactionType;
        previewRow.sourceRowKey = parsedRow.sourceRowKey.trimmed();
        previewRow.detail = parsedRow.detail;

        if (previewRow.postedDate.isValid() && !previewRow.description.isEmpty() && previewRow.amount > 0.0) {
            previewRow.fingerprint = buildImportFingerprint(previewRow.postedDate,
                                                            previewRow.amount,
                                                            previewRow.transactionType,
                                                            previewRow.description,
                                                            previewRow.memo);
            if (previewRow.sourceRowKey.isEmpty()) {
                previewRow.sourceRowKey = previewRow.fingerprint;
            }
        }

        if (!previewRow.postedDate.isValid()) {
            previewRow.status = ImportRowStatus::Invalid;
            previewRow.detail = "Missing or invalid date";
        } else if (previewRow.description.isEmpty()) {
            previewRow.status = ImportRowStatus::Invalid;
            previewRow.detail = "Missing description";
        } else if (previewRow.amount <= 0.0) {
            previewRow.status = ImportRowStatus::Invalid;
            previewRow.detail = "Missing or invalid amount";
        } else if (lookup.transactionIdBySourceRowKey.contains(previewRow.sourceRowKey)) {
            previewRow.status = ImportRowStatus::Duplicate;
            previewRow.matchedTransactionId = lookup.transactionIdBySourceRowKey.value(previewRow.sourceRowKey);
            previewRow.detail = "Already imported for this account";
        } else if (lookup.transactionIdByFingerprint.contains(previewRow.fingerprint)) {
            previewRow.status = ImportRowStatus::Duplicate;
            previewRow.matchedTransactionId = lookup.transactionIdByFingerprint.value(previewRow.fingerprint);
            previewRow.detail = "Matches a previously imported transaction";
        } else if (const auto matchedTransactionId = m_transactionRepository.findPotentialDuplicate(targetAccountId,
                                                                                                   previewRow.postedDate,
                                                                                                   previewRow.amount,
                                                                                                   previewRow.transactionType,
                                                                                                   normalizeImportedText(previewRow.description));
                   matchedTransactionId.has_value()) {
            previewRow.status = ImportRowStatus::Duplicate;
            previewRow.matchedTransactionId = matchedTransactionId;
            previewRow.detail = "Matches an existing transaction";
        } else {
            previewRow.status = ImportRowStatus::Ready;
            previewRow.selected = true;
            previewRow.detail = "Ready to import";
        }

        preview.rows.append(previewRow);
    }

    if (preview.rows.isEmpty()) {
        result.errorMessage = "No transactions were found in the statement file.";
        return result;
    }

    updatePreviewCounts(&preview);

    result.success = true;
    result.preview = preview;
    return result;
}

ImportCommitResult FinanceService::commitStatementImport(const ImportCommitRequest& request) const
{
    ImportCommitResult result;
    const ImportPreview& preview = request.preview;

    if (!m_accountRepository.findById(preview.targetAccountId).has_value()) {
        result.errorMessage = "The destination account no longer exists.";
        return result;
    }

    QSqlDatabase database = m_statementImportRepository.database();
    if (!database.transaction()) {
        result.errorMessage = "Couldn't start the import transaction.";
        return result;
    }

    const int importId = m_statementImportRepository.createImportSession(preview.targetAccountId,
                                                                         preview.institution,
                                                                         preview.sourceFormat,
                                                                         preview.sourceFileName,
                                                                         preview.sourceFileHash,
                                                                         QDateTime::currentDateTimeUtc());
    if (importId <= 0) {
        database.rollback();
        result.errorMessage = "Couldn't save the import session.";
        return result;
    }

    QList<Transaction> transactionsToCreate;
    for (const ImportPreviewRow& row : preview.rows) {
        if (row.status == ImportRowStatus::Ready && row.selected) {
            transactionsToCreate.append(importPreviewRowToTransaction(row, preview.targetAccountId));
        }
    }

    const QList<int> createdTransactionIds = m_transactionRepository.createMany(transactionsToCreate);
    if (createdTransactionIds.size() != transactionsToCreate.size()) {
        database.rollback();
        result.errorMessage = "Couldn't create the imported transactions.";
        return result;
    }

    int createdIndex = 0;
    for (const ImportPreviewRow& row : preview.rows) {
        ImportRowStatus finalStatus = row.status;
        std::optional<int> matchedTransactionId = row.matchedTransactionId;
        std::optional<int> createdTransactionId = row.createdTransactionId;

        switch (row.status) {
        case ImportRowStatus::Ready:
            if (row.selected) {
                finalStatus = ImportRowStatus::Imported;
                createdTransactionId = createdTransactionIds.at(createdIndex++);
                ++result.importedCount;
            } else {
                finalStatus = ImportRowStatus::Skipped;
                ++result.skippedCount;
            }
            break;
        case ImportRowStatus::Duplicate:
            ++result.duplicateCount;
            break;
        case ImportRowStatus::Invalid:
            ++result.invalidCount;
            break;
        case ImportRowStatus::Skipped:
            ++result.skippedCount;
            break;
        case ImportRowStatus::Imported:
            finalStatus = ImportRowStatus::Imported;
            ++result.importedCount;
            break;
        }

        if (!m_statementImportRepository.createImportRow(importId,
                                                         row,
                                                         finalStatus,
                                                         matchedTransactionId,
                                                         createdTransactionId)) {
            database.rollback();
            result.errorMessage = "Couldn't save the import row metadata.";
            return result;
        }
    }

    if (!database.commit()) {
        database.rollback();
        result.errorMessage = "Couldn't finalize the import.";
        return result;
    }

    result.success = true;
    result.importId = importId;
    return result;
}

QString extractTagValue(const QString& block, const QString& tag)
{
    const QRegularExpression expression(QStringLiteral("<%1>([^<\\r\\n]*)").arg(tag),
                                        QRegularExpression::CaseInsensitiveOption);
    const QRegularExpressionMatch match = expression.match(block);
    return match.hasMatch() ? match.captured(1).trimmed() : QString();
}

QList<QString> extractTransactionBlocks(const QString& content)
{
    QList<QString> blocks;

    QRegularExpression expression("<STMTTRN>(.*?)</STMTTRN>",
                                  QRegularExpression::CaseInsensitiveOption | QRegularExpression::DotMatchesEverythingOption);
    QRegularExpressionMatchIterator iterator = expression.globalMatch(content);
    while (iterator.hasNext()) {
        blocks.append(iterator.next().captured(1));
    }

    if (!blocks.isEmpty()) {
        return blocks;
    }

    const QStringList segments = content.split(QRegularExpression("<STMTTRN>", QRegularExpression::CaseInsensitiveOption),
                                               Qt::SkipEmptyParts);
    for (const QString& segment : segments) {
        QString block = segment;
        const QRegularExpression endExpression("</STMTTRN>|</BANKTRANLIST>", QRegularExpression::CaseInsensitiveOption);
        const QRegularExpressionMatch match = endExpression.match(block);
        if (match.hasMatch()) {
            block = block.left(match.capturedStart()).trimmed();
        }
        if (!block.isEmpty()) {
            blocks.append(block);
        }
    }

    return blocks;
}

std::optional<TransactionType> transactionTypeFromOfx(const QString& value)
{
    const QString normalized = normalizeImportedText(value);
    if (normalized.contains("xfer") || normalized.contains("transfer")) {
        return TransactionType::Transfer;
    }
    if (normalized.contains("credit") || normalized.contains("dep") || normalized.contains("directdep")) {
        return TransactionType::Income;
    }
    if (normalized.contains("debit") || normalized.contains("payment") || normalized.contains("atm") || normalized.contains("pos") || normalized.contains("check")) {
        return TransactionType::Expense;
    }
    return std::nullopt;
}

StatementParseResult OfxQfxParser::parse(const QString& filePath, const QByteArray& contents) const
{
    StatementParseResult result;
    result.sourceFormat = "OFX/QFX";
    result.sourceFileName = QFileInfo(filePath).fileName();

    const QString text = QString::fromUtf8(contents);
    const QList<QString> blocks = extractTransactionBlocks(text);
    if (blocks.isEmpty()) {
        result.errorMessage = "No transactions were found in the OFX/QFX file.";
        return result;
    }

    for (const QString& block : blocks) {
        StatementParseRow row;
        QStringList issues;

        const auto postedDate = parseFlexibleImportDate(extractTagValue(block, "DTPOSTED"));
        if (postedDate.has_value()) {
            row.postedDate = *postedDate;
        } else {
            issues.append("Missing or invalid date");
        }

        row.description = extractTagValue(block, "NAME");
        const QString memo = extractTagValue(block, "MEMO");
        if (row.description.isEmpty()) {
            row.description = memo;
        }
        if (!memo.isEmpty() && normalizeImportedText(memo) != normalizeImportedText(row.description)) {
            row.memo = memo;
        }

        if (row.description.isEmpty()) {
            issues.append("Missing description");
        }

        double signedAmount = 0.0;
        if (!parseCurrencyAmount(extractTagValue(block, "TRNAMT"), &signedAmount)) {
            issues.append("Missing or invalid amount");
        } else {
            row.amount = qAbs(signedAmount);
        }

        const auto explicitType = transactionTypeFromOfx(extractTagValue(block, "TRNTYPE"));
        if (explicitType.has_value()) {
            row.transactionType = *explicitType;
        } else if (signedAmount < 0.0) {
            row.transactionType = TransactionType::Expense;
        } else if (signedAmount > 0.0) {
            row.transactionType = TransactionType::Income;
        }

        row.sourceRowKey = extractTagValue(block, "FITID");
        row.detail = issues.join("; ");
        result.rows.append(row);
    }

    result.success = true;
    return result;
}

QList<QStringList> parseCsvRows(const QString& text)
{
    QList<QStringList> rows;
    QStringList currentRow;
    QString currentField;
    bool inQuotes = false;

    for (int index = 0; index < text.size(); ++index) {
        const QChar character = text.at(index);

        if (character == '"') {
            const bool escapedQuote = inQuotes && index + 1 < text.size() && text.at(index + 1) == '"';
            if (escapedQuote) {
                currentField.append('"');
                ++index;
            } else {
                inQuotes = !inQuotes;
            }
            continue;
        }

        if (!inQuotes && character == ',') {
            currentRow.append(currentField);
            currentField.clear();
            continue;
        }

        if (!inQuotes && (character == '\n' || character == '\r')) {
            if (character == '\r' && index + 1 < text.size() && text.at(index + 1) == '\n') {
                ++index;
            }

            currentRow.append(currentField);
            currentField.clear();
            if (!(currentRow.size() == 1 && currentRow.first().trimmed().isEmpty())) {
                rows.append(currentRow);
            }
            currentRow.clear();
            continue;
        }

        currentField.append(character);
    }

    if (!currentField.isEmpty() || !currentRow.isEmpty()) {
        currentRow.append(currentField);
        rows.append(currentRow);
    }

    if (!rows.isEmpty() && !rows.first().isEmpty()) {
        rows[0][0].remove(QChar(0xfeff));
    }

    return rows;
}

QString normalizeHeader(const QString& value)
{
    QString normalized = normalizeImportedText(value);
    normalized.remove(QRegularExpression("[^a-z0-9]"));
    return normalized;
}

int findHeaderIndex(const QStringList& headers, const QStringList& aliases)
{
    for (int index = 0; index < headers.size(); ++index) {
        const QString normalized = normalizeHeader(headers.at(index));
        for (const QString& alias : aliases) {
            if (normalized == normalizeHeader(alias)) {
                return index;
            }
        }
    }
    return -1;
}

std::optional<TransactionType> transactionTypeFromText(const QString& value)
{
    const QString normalized = normalizeImportedText(value);
    if (normalized.contains("transfer") || normalized.contains("xfer")) {
        return TransactionType::Transfer;
    }
    if (normalized.contains("credit") || normalized.contains("deposit") || normalized.contains("income")) {
        return TransactionType::Income;
    }
    if (normalized.contains("debit") || normalized.contains("payment") || normalized.contains("purchase") || normalized.contains("withdrawal") || normalized.contains("fee")) {
        return TransactionType::Expense;
    }
    return std::nullopt;
}

QString fieldValue(const QStringList& row, int index)
{
    return index >= 0 && index < row.size() ? row.at(index).trimmed() : QString();
}

StatementParseResult WellsFargoCsvParser::parse(const QString& filePath, const QByteArray& contents) const
{
    StatementParseResult result;
    result.sourceFormat = "CSV";
    result.sourceFileName = QFileInfo(filePath).fileName();

    const QList<QStringList> rows = parseCsvRows(QString::fromUtf8(contents));
    if (rows.size() < 2) {
        result.errorMessage = "The CSV file does not contain a header row and transaction rows.";
        return result;
    }

    const QStringList headers = rows.first();
    const int dateIndex = findHeaderIndex(headers, {"Date", "Posted Date", "Transaction Date"});
    const int descriptionIndex = findHeaderIndex(headers, {"Description", "Details", "Name", "Transaction Description"});
    const int memoIndex = findHeaderIndex(headers, {"Memo", "Notes", "Note"});
    const int amountIndex = findHeaderIndex(headers, {"Amount", "Transaction Amount"});
    const int debitIndex = findHeaderIndex(headers, {"Debit", "Withdrawal", "Withdrawals"});
    const int creditIndex = findHeaderIndex(headers, {"Credit", "Deposit", "Deposits"});
    const int typeIndex = findHeaderIndex(headers, {"Type", "Transaction Type"});
    const int sourceIdIndex = findHeaderIndex(headers, {"FITID", "Transaction Id", "Transaction ID", "Reference", "Reference Number", "Confirmation Number"});

    if (dateIndex < 0 || descriptionIndex < 0 || (amountIndex < 0 && debitIndex < 0 && creditIndex < 0)) {
        result.errorMessage = "The CSV file is missing one of the required Wells Fargo columns: date, description, or amount.";
        return result;
    }

    for (int rowIndex = 1; rowIndex < rows.size(); ++rowIndex) {
        const QStringList rowValues = rows.at(rowIndex);
        if (rowValues.isEmpty()) {
            continue;
        }

        StatementParseRow row;
        QStringList issues;

        const auto postedDate = parseFlexibleImportDate(fieldValue(rowValues, dateIndex));
        if (postedDate.has_value()) {
            row.postedDate = *postedDate;
        } else {
            issues.append("Missing or invalid date");
        }

        row.description = fieldValue(rowValues, descriptionIndex);
        if (row.description.isEmpty()) {
            issues.append("Missing description");
        }

        const QString memo = fieldValue(rowValues, memoIndex);
        if (!memo.isEmpty()) {
            row.memo = memo;
        }

        double signedAmount = 0.0;
        bool amountOk = false;

        const QString amountText = fieldValue(rowValues, amountIndex);
        if (!amountText.isEmpty()) {
            amountOk = parseCurrencyAmount(amountText, &signedAmount);
        } else {
            double creditValue = 0.0;
            double debitValue = 0.0;
            const bool hasCredit = parseCurrencyAmount(fieldValue(rowValues, creditIndex), &creditValue);
            const bool hasDebit = parseCurrencyAmount(fieldValue(rowValues, debitIndex), &debitValue);

            if (hasCredit && qAbs(creditValue) > 0.0) {
                signedAmount = qAbs(creditValue);
                amountOk = true;
            } else if (hasDebit && qAbs(debitValue) > 0.0) {
                signedAmount = -qAbs(debitValue);
                amountOk = true;
            }
        }

        if (!amountOk) {
            issues.append("Missing or invalid amount");
        } else {
            row.amount = qAbs(signedAmount);
        }

        const auto explicitType = transactionTypeFromText(fieldValue(rowValues, typeIndex));
        if (explicitType.has_value()) {
            row.transactionType = *explicitType;
        } else if (signedAmount < 0.0) {
            row.transactionType = TransactionType::Expense;
        } else if (signedAmount > 0.0) {
            row.transactionType = TransactionType::Income;
        }

        row.sourceRowKey = fieldValue(rowValues, sourceIdIndex);
        row.detail = issues.join("; ");
        result.rows.append(row);
    }

    if (result.rows.isEmpty()) {
        result.errorMessage = "No transactions were found in the CSV file.";
        return result;
    }

    result.success = true;
    return result;
}

ImportCommitResult ImportStatementDialog::result() const
{
    return m_result;
}

void ImportStatementDialog::browseForFile()
{
    const QString path = QFileDialog::getOpenFileName(this,
                                                      "Choose statement file",
                                                      QString(),
                                                      "Statement files (*.csv *.ofx *.qfx);;CSV files (*.csv);;OFX/QFX files (*.ofx *.qfx);;All files (*.*)");
    if (!path.isEmpty()) {
        m_filePathEdit->setText(path);
    }
}

void ImportStatementDialog::createAccount()
{
    AccountDialog dialog(this);
    if (dialog.exec() != QDialog::Accepted) {
        return;
    }

    const int accountId = m_financeService->createAccount(dialog.account());
    if (accountId <= 0) {
        QMessageBox::warning(this, "Create account", "The new account couldn't be created.");
        return;
    }

    loadAccounts(accountId);
    emit accountsChanged();
}

void ImportStatementDialog::goBack()
{
    const int currentIndex = m_steps->currentIndex();
    if (currentIndex > 0) {
        m_steps->setCurrentIndex(currentIndex - 1);
        updateButtons();
    }
}

void ImportStatementDialog::goNext()
{
    const int currentIndex = m_steps->currentIndex();

    if (currentIndex == 0) {
        if (m_filePathEdit->text().trimmed().isEmpty()) {
            QMessageBox::information(this, "Import statement", "Choose a Wells Fargo statement file first.");
            return;
        }

        m_steps->setCurrentIndex(1);
        updateButtons();
        return;
    }

    if (currentIndex == 1) {
        const int accountId = m_accountCombo->currentData().toInt();
        if (accountId <= 0) {
            QMessageBox::information(this, "Import statement", "Choose or create a destination account.");
            return;
        }

        const ImportPreviewResult previewResult = m_financeService->previewStatementImport(m_filePathEdit->text().trimmed(), accountId);
        if (!previewResult.success || !previewResult.preview.has_value()) {
            QMessageBox::warning(this, "Preview import", previewResult.errorMessage);
            return;
        }

        m_preview = *previewResult.preview;
        refreshPreviewTable();
        m_steps->setCurrentIndex(2);
        updateButtons();
    }
}

void ImportStatementDialog::importSelected()
{
    ImportCommitRequest request;
    request.preview = m_preview;

    m_result = m_financeService->commitStatementImport(request);
    if (!m_result.success) {
        QMessageBox::warning(this, "Import statement", m_result.errorMessage);
        return;
    }

    QMessageBox::information(this,
                             "Import statement",
                             QStringLiteral("Imported %1 transactions. %2 duplicates, %3 invalid rows, %4 skipped rows.")
                                 .arg(m_result.importedCount)
                                 .arg(m_result.duplicateCount)
                                 .arg(m_result.invalidCount)
                                 .arg(m_result.skippedCount));
    accept();
}

void ImportStatementDialog::loadAccounts(int preferredAccountId)
{
    const QList<Account> accounts = m_financeService->accounts();
    m_accountCombo->blockSignals(true);
    m_accountCombo->clear();
    for (const Account& account : accounts) {
        m_accountCombo->addItem(account.name, account.id);
    }

    if (preferredAccountId > 0) {
        const int index = m_accountCombo->findData(preferredAccountId);
        if (index >= 0) {
            m_accountCombo->setCurrentIndex(index);
        }
    }

    m_accountCombo->blockSignals(false);
}

void ImportStatementDialog::refreshPreviewTable()
{
    m_previewTable->blockSignals(true);
    m_previewTable->setRowCount(m_preview.rows.size());

    for (int rowIndex = 0; rowIndex < m_preview.rows.size(); ++rowIndex) {
        const ImportPreviewRow& row = m_preview.rows.at(rowIndex);

        auto* importItem = new QTableWidgetItem();
        importItem->setData(Qt::UserRole, rowIndex);
        if (row.status == ImportRowStatus::Ready) {
            importItem->setFlags(importItem->flags() | Qt::ItemIsUserCheckable | Qt::ItemIsEnabled);
            importItem->setCheckState(row.selected ? Qt::Checked : Qt::Unchecked);
        } else {
            importItem->setFlags(Qt::ItemIsEnabled);
            importItem->setCheckState(Qt::Unchecked);
        }

        m_previewTable->setItem(rowIndex, 0, importItem);
        m_previewTable->setItem(rowIndex, 1, new QTableWidgetItem(toDisplayString(row.status)));
        m_previewTable->setItem(rowIndex, 2, new QTableWidgetItem(row.postedDate.isValid() ? row.postedDate.toString(Qt::ISODate) : QString()));
        m_previewTable->setItem(rowIndex, 3, new QTableWidgetItem(row.description));
        m_previewTable->setItem(rowIndex, 4, new QTableWidgetItem(toDisplayString(row.transactionType)));
        m_previewTable->setItem(rowIndex, 5, new QTableWidgetItem(QString("$%1").arg(row.amount, 0, 'f', 2)));
        m_previewTable->setItem(rowIndex, 6, new QTableWidgetItem(row.detail));
    }

    m_previewTable->blockSignals(false);

    m_previewSummaryLabel->setText(
        QStringLiteral("%1 -> %2 (%3)\nReady: %4, duplicates: %5, invalid: %6")
            .arg(m_preview.sourceFileName,
                 m_accountCombo->currentText(),
                 m_preview.sourceFormat,
                 QString::number(m_preview.readyCount),
                 QString::number(m_preview.duplicateCount),
                 QString::number(m_preview.invalidCount)));
}

void ImportStatementDialog::updateButtons()
{
    const int currentIndex = m_steps->currentIndex();
    m_backButton->setVisible(currentIndex > 0);
    m_nextButton->setVisible(currentIndex < 2);
    m_importButton->setVisible(currentIndex == 2);
}

void ImportStatementDialog::updatePreviewSelection(QTableWidgetItem* item)
{
    if (item == nullptr || item->column() != 0) {
        return;
    }

    const int rowIndex = item->data(Qt::UserRole).toInt();
    if (rowIndex < 0 || rowIndex >= m_preview.rows.size()) {
        return;
    }

    ImportPreviewRow& row = m_preview.rows[rowIndex];
    if (row.status == ImportRowStatus::Ready) {
        row.selected = item->checkState() == Qt::Checked;
    }
}

void CalendarPage::loadSelectedDate()
{
    const QList<CalendarEvent> events = m_calendarService->eventsForDate(m_calendar->selectedDate());
    m_eventList->clear();
    if (events.isEmpty()) {
        auto* placeholder = new QListWidgetItem("No events scheduled for this date.");
        placeholder->setFlags(placeholder->flags() & ~(Qt::ItemIsSelectable | Qt::ItemIsEnabled));
        m_eventList->addItem(placeholder);
        return;
    }

    for (const CalendarEvent& event : events) {
        auto* item = new QListWidgetItem(QString("%1  •  %2")
                                             .arg(event.startDateTime.time().toString("h:mm AP"))
                                             .arg(event.title));
        item->setData(Qt::UserRole, event.id);
        m_eventList->addItem(item);
    }
}

void CalendarPage::createEvent()
{
    EventDialog dialog(this);
    CalendarEvent event;
    event.startDateTime = QDateTime(m_calendar->selectedDate(), QTime(9, 0));
    event.endDateTime = QDateTime(m_calendar->selectedDate(), QTime(10, 0));
    dialog.setEvent(event);
    if (dialog.exec() == QDialog::Accepted) {
        m_calendarService->createEvent(dialog.event());
        loadSelectedDate();
        emit dataChanged();
    }
}

void CalendarPage::editSelectedEvent()
{
    const auto* item = m_eventList->currentItem();
    if (item == nullptr) {
        return;
    }

    const int id = item->data(Qt::UserRole).toInt();
    const QList<CalendarEvent> events = m_calendarService->events();
    const auto match = std::find_if(events.begin(), events.end(), [id](const CalendarEvent& event) { return event.id == id; });
    if (match == events.end()) {
        return;
    }

    EventDialog dialog(this);
    dialog.setEvent(*match);
    if (dialog.exec() == QDialog::Accepted) {
        m_calendarService->updateEvent(dialog.event());
        loadSelectedDate();
        emit dataChanged();
    }
}

void CalendarPage::deleteSelectedEvent()
{
    const auto* item = m_eventList->currentItem();
    if (item == nullptr) {
        return;
    }

    if (QMessageBox::question(this, "Delete event", "Delete the selected event?") == QMessageBox::Yes) {
        m_calendarService->deleteEvent(item->data(Qt::UserRole).toInt());
        loadSelectedDate();
        emit dataChanged();
    }
}

void FinancePage::refresh()
{
    const FinanceSummary summary = m_financeService->summary();
    const QList<Account> accounts = m_financeService->accounts();
    const QList<Transaction> allTransactions = m_financeService->transactions();

    m_incomeCard->setValue(QString("$%1").arg(summary.currentMonthIncome, 0, 'f', 2));
    m_incomeCard->setSubtitle("Income recorded this month");
    m_expenseCard->setValue(QString("$%1").arg(summary.currentMonthExpenses, 0, 'f', 2));
    m_expenseCard->setSubtitle("Expenses recorded this month");

    m_accountTree->clear();
    for (const Account& account : summary.accounts) {
        auto* item = new QTreeWidgetItem({account.name, QString("$%1").arg(account.currentBalance, 0, 'f', 2)});
        item->setData(0, Qt::UserRole, account.id);
        m_accountTree->addTopLevelItem(item);
    }

    m_breakdownTree->clear();
    for (auto it = summary.expenseByCategory.cbegin(); it != summary.expenseByCategory.cend(); ++it) {
        m_breakdownTree->addTopLevelItem(new QTreeWidgetItem({it.key(), QString("$%1").arg(it.value(), 0, 'f', 2)}));
    }

    m_accountFilter->blockSignals(true);
    const int currentAccountId = m_accountFilter->currentData().toInt();
    m_accountFilter->clear();
    m_accountFilter->addItem("All accounts", -1);
    for (const Account& account : accounts) {
        m_accountFilter->addItem(account.name, account.id);
    }
    const int accountIndex = m_accountFilter->findData(currentAccountId);
    m_accountFilter->setCurrentIndex(accountIndex >= 0 ? accountIndex : 0);
    m_accountFilter->blockSignals(false);

    QList<Transaction> transactions = allTransactions;
    const int accountFilter = m_accountFilter->currentData().toInt();
    const int typeFilter = m_typeFilter->currentData().toInt();
    transactions.erase(std::remove_if(transactions.begin(), transactions.end(), [accountFilter, typeFilter](const Transaction& transaction) {
                           if (accountFilter >= 0 && transaction.accountId != accountFilter) {
                               return true;
                           }
                           if (typeFilter >= 0 && static_cast<int>(transaction.type) != typeFilter) {
                               return true;
                           }
                           return false;
                       }),
                       transactions.end());

    m_transactionTable->setRowCount(transactions.size());
    for (int row = 0; row < transactions.size(); ++row) {
        const Transaction& transaction = transactions.at(row);
        const auto account = std::find_if(accounts.begin(), accounts.end(), [&transaction](const Account& account) { return account.id == transaction.accountId; });
        auto* dateItem = new QTableWidgetItem(QLocale().toString(transaction.transactionDate, QLocale::ShortFormat));
        dateItem->setData(Qt::UserRole, transaction.id);
        m_transactionTable->setItem(row, 0, dateItem);
        m_transactionTable->setItem(row, 1, new QTableWidgetItem(transaction.description));
        m_transactionTable->setItem(row, 2, new QTableWidgetItem(account != accounts.end() ? account->name : QString("Unknown")));
        m_transactionTable->setItem(row, 3, new QTableWidgetItem(toDisplayString(transaction.type)));
        m_transactionTable->setItem(row, 4, new QTableWidgetItem(transaction.category));
        m_transactionTable->setItem(row, 5, new QTableWidgetItem(QString("$%1").arg(transaction.amount, 0, 'f', 2)));
    }
}

    }

    const QList<Account> accounts = m_financeService->accounts();

void FinancePage::deleteSelectedAccount()
{
    auto* item = m_accountTree->currentItem();
    if (item == nullptr) {
        return;

    }

    tasks.erase(std::remove_if(tasks.begin(), tasks.end(), [statusFilter](const Task& task) {
                    return statusFilter >= 0 && static_cast<int>(task.status) != statusFilter;
                }),
                tasks.end());

    if (m_sortCombo->currentIndex() == 1) {
        std::sort(tasks.begin(), tasks.end(), [](const Task& left, const Task& right) {
            return static_cast<int>(left.priority) > static_cast<int>(right.priority);
        });
    } else {
        std::sort(tasks.begin(), tasks.end(), [](const Task& left, const Task& right) {
            return left.dueDate.value_or(QDate(2999, 12, 31)) < right.dueDate.value_or(QDate(2999, 12, 31));
        });
    }

    m_table->setRowCount(tasks.size());
    for (int row = 0; row < tasks.size(); ++row) {
        const Task& task = tasks.at(row);
        auto* titleItem = new QTableWidgetItem(task.title);
        titleItem->setData(Qt::UserRole, task.id);
        m_table->setItem(row, 0, titleItem);
        m_table->setItem(row, 1, new QTableWidgetItem(toDisplayString(task.status)));
        m_table->setItem(row, 2, new QTableWidgetItem(toDisplayString(task.priority)));
        m_table->setItem(row, 3, new QTableWidgetItem(task.dueDate.has_value() ? QLocale().toString(*task.dueDate, QLocale::ShortFormat) : "-"));
        m_table->setItem(row, 4, new QTableWidgetItem(task.description));
    }
}

void TasksPage::createTask()
{
    TaskDialog dialog(this);
    if (dialog.exec() == QDialog::Accepted) {
        m_taskService->createTask(dialog.task());
        refresh();
        emit dataChanged();
    }
}

void TasksPage::editSelectedTask()
{
    const auto task = selectedTask();
    if (!task.has_value()) {
        return;
    }

    TaskDialog dialog(this);
    dialog.setTask(*task);
    if (dialog.exec() == QDialog::Accepted) {
        m_taskService->updateTask(dialog.task());
        refresh();
        emit dataChanged();
    }
}

void TasksPage::deleteSelectedTask()
{
    const auto task = selectedTask();
    if (!task.has_value()) {
        return;
    }

    if (QMessageBox::question(this, "Delete task", "Delete the selected task?") == QMessageBox::Yes) {
        m_taskService->deleteTask(task->id);
        refresh();
        emit dataChanged();
    }
}

void TasksPage::markSelectedComplete()
{
    const auto task = selectedTask();
    if (!task.has_value()) {
        return;
    }

    m_taskService->markCompleted(task->id);
    refresh();
    emit dataChanged();
}

std::optional<Task> TasksPage::selectedTask() const
{
    const auto items = m_table->selectedItems();
    if (items.isEmpty()) {
        return std::nullopt;
    }

    const int id = items.first()->data(Qt::UserRole).toInt();
    const QList<Task> tasks = m_taskService->tasks();
    const auto match = std::find_if(tasks.begin(), tasks.end(), [id](const Task& task) { return task.id == id; });
    if (match == tasks.end()) {
        return std::nullopt;
    }
    return *match;
}

bool MainWindow::eventFilter(QObject* watched, QEvent* event)
{
    if (event->type() == QEvent::KeyPress) {
        auto* keyEvent = static_cast<QKeyEvent*>(event);
        if (!keyEvent->isAutoRepeat() && keyEvent->key() == Qt::Key_Escape && keyEvent->modifiers() == Qt::NoModifier) {
            if (QApplication::activeModalWidget() != nullptr && QApplication::activeModalWidget() != this) {
                return QMainWindow::eventFilter(watched, event);
            }

            auto* widget = qobject_cast<QWidget*>(watched);
            if (widget != nullptr && widget->window() == this && isActiveWindow()) {
                handleEscapeShortcut();
                return true;
            }
        } else if (!keyEvent->isAutoRepeat()) {
            m_escapePending = false;
        }
    }

    return QMainWindow::eventFilter(watched, event);
}

void MainWindow::handleEscapeShortcut()
{
    if (m_escapePending && m_escapeTimer.isValid() && m_escapeTimer.elapsed() <= 500) {
        m_escapePending = false;
        if (m_currentPage == AppPage::Home) {
            requestSignOut();
        } else {
            setCurrentPage(AppPage::Home);
        }
        return;
    }

    m_escapePending = true;
    m_escapeTimer.restart();
}

{
    m_escapePending = false;
    if (QMessageBox::question(this, "Sign Out", "Are you sure you want to sign out?") == QMessageBox::Yes) {
        close();
    }
}

QString writeStatementFile(const QString& extension, const QByteArray& content)
{
    const QString directoryPath = QDir(QDir::tempPath()).filePath("lifeplanner_statement_tests");
    QDir directory;
    if (!directory.mkpath(directoryPath)) {
        return {};
    }

    const QString path = QDir(directoryPath).filePath(QString("statement_%1.%2").arg(QUuid::createUuid().toString(QUuid::WithoutBraces), extension));

    QFile file(path);
    if (!file.open(QIODevice::WriteOnly | QIODevice::Truncate)) {
        return {};
    }

    file.write(content);
    file.close();
    return path;
}

int queryCount(const QSqlDatabase& database, const QString& sql)
{
    QSqlQuery query(database);
    if (!query.exec(sql) || !query.next()) {
        return -1;
    }

    return query.value(0).toInt();
}

double accountBalance(FinanceService* financeService, int accountId)
{
    const QList<Account> accounts = financeService->accounts();
    const auto match = std::find_if(accounts.begin(), accounts.end(), [accountId](const Account& account) { return account.id == accountId; });
    return match != accounts.end() ? match->currentBalance : 0.0;
}

    account.type = AccountType::Checking;
    const int accountId = context->financeService()->createAccount(account);
    QVERIFY(accountId > 0);

    const double startingBalance = accountBalance(context->financeService(), accountId);
    const int startingTransactions = context->financeService()->transactions().size();

    const QString csvPath = writeStatementFile(
        "csv",
        QByteArray(
            "Date,Description,Memo,Amount,Type\r\n"
            "04/08/2026,Side Hustle,,125.00,Credit\r\n"
            "04/09/2026,Concert Tickets,,55.00,Debit\r\n"));
    QVERIFY(!csvPath.isEmpty());

    ImportPreviewResult previewResult = context->financeService()->previewStatementImport(csvPath, accountId);
    QVERIFY(previewResult.success);
    QVERIFY(previewResult.preview.has_value());

    ImportPreview preview = *previewResult.preview;
    QCOMPARE(preview.readyCount, 2);
    preview.rows[1].selected = false;

    ImportCommitRequest request;
    request.preview = preview;

    const ImportCommitResult commitResult = context->financeService()->commitStatementImport(request);
    QVERIFY(commitResult.success);
    QCOMPARE(commitResult.importedCount, 1);
    QCOMPARE(commitResult.skippedCount, 1);
    QCOMPARE(commitResult.duplicateCount, 0);
    QCOMPARE(commitResult.invalidCount, 0);

    QCOMPARE(context->financeService()->transactions().size(), startingTransactions + 1);
    QVERIFY(qAbs(accountBalance(context->financeService(), accountId) - (startingBalance + 125.0)) < 0.001);

    const QSqlDatabase database = context->databaseManager()->database();
    QCOMPARE(queryCount(database, "SELECT COUNT(*) FROM statement_imports"), 1);
    QCOMPARE(queryCount(database, QString("SELECT COUNT(*) FROM statement_import_rows WHERE status = %1").arg(static_cast<int>(ImportRowStatus::Skipped))), 1);
    QCOMPARE(queryCount(database, "SELECT COUNT(*) FROM statement_import_rows WHERE created_transaction_id IS NOT NULL"), 1);
}

void LifePlannerTests::statementImportDeduplicatesAndPersistsMetadata()
{
    auto context = lifeplanner::tests::createTestContext();

    Account account;
    account.name = "Dedup Import";
    account.type = AccountType::Checking;
    const int accountId = context->financeService()->createAccount(account);
    QVERIFY(accountId > 0);

    const int startingTransactions = context->financeService()->transactions().size();

    const QString csvPath = writeStatementFile(
        "csv",
        QByteArray(
            "Date,Description,Memo,Amount,Type\r\n"
            "04/10/2026,Client Payment,,300.00,Credit\r\n"
            "04/11/2026,Hardware Store,,42.10,Debit\r\n"
            ",Broken Row,,12.00,Debit\r\n"));
    QVERIFY(!csvPath.isEmpty());

    ImportPreviewResult firstPreviewResult = context->financeService()->previewStatementImport(csvPath, accountId);
    QVERIFY(firstPreviewResult.success);
    QVERIFY(firstPreviewResult.preview.has_value());
    QCOMPARE(firstPreviewResult.preview->readyCount, 2);
    QCOMPARE(firstPreviewResult.preview->invalidCount, 1);

    ImportCommitRequest firstRequest;
    firstRequest.preview = *firstPreviewResult.preview;

    const ImportCommitResult firstCommitResult = context->financeService()->commitStatementImport(firstRequest);
    QVERIFY(firstCommitResult.success);
    QCOMPARE(firstCommitResult.importedCount, 2);
    QCOMPARE(firstCommitResult.invalidCount, 1);
    QCOMPARE(firstCommitResult.duplicateCount, 0);
    QCOMPARE(context->financeService()->transactions().size(), startingTransactions + 2);

    const ImportPreviewResult secondPreviewResult = context->financeService()->previewStatementImport(csvPath, accountId);
    QVERIFY(secondPreviewResult.success);
    QVERIFY(secondPreviewResult.preview.has_value());
    QCOMPARE(secondPreviewResult.preview->readyCount, 0);
    QCOMPARE(secondPreviewResult.preview->duplicateCount, 2);
    QCOMPARE(secondPreviewResult.preview->invalidCount, 1);

    ImportCommitRequest secondRequest;
    secondRequest.preview = *secondPreviewResult.preview;

    const ImportCommitResult secondCommitResult = context->financeService()->commitStatementImport(secondRequest);
    QVERIFY(secondCommitResult.success);
    QCOMPARE(secondCommitResult.importedCount, 0);
    QCOMPARE(secondCommitResult.duplicateCount, 2);
    QCOMPARE(secondCommitResult.invalidCount, 1);
    QCOMPARE(context->financeService()->transactions().size(), startingTransactions + 2);

    const QSqlDatabase database = context->databaseManager()->database();
    QCOMPARE(queryCount(database, "SELECT COUNT(*) FROM statement_imports"), 2);
    QCOMPARE(queryCount(database, "SELECT COUNT(*) FROM statement_import_rows"), 6);
    QCOMPARE(queryCount(database, QString("SELECT COUNT(*) FROM statement_import_rows WHERE status = %1").arg(static_cast<int>(ImportRowStatus::Imported))), 2);
    QCOMPARE(queryCount(database, QString("SELECT COUNT(*) FROM statement_import_rows WHERE status = %1").arg(static_cast<int>(ImportRowStatus::Duplicate))), 2);
    QCOMPARE(queryCount(database, QString("SELECT COUNT(*) FROM statement_import_rows WHERE status = %1").arg(static_cast<int>(ImportRowStatus::Invalid))), 2);
}
